一、vue的安装方法

  1. 方法一：直接CDN引入

  2. 方法二：下载和引入
    开发环境：https://cn.vuejs.org/js/vue.js
    生产环境：https://cn.vuejs.org/js/vue.min.js

  3. NPM安装
    命令行工具 (CLI)
    Vue.js 有一款官方的脚手架生成工具 vue-cli
    1) 全局安装
    npm install –g vue-cli
    2) 创建项目
    vue init webpack myproject
    3) 安装依赖包
    npm install
    4) 启动服务：
    正常开发时，就会运行命令npm run dev，启动一个小型的express 服务。

五、事件修饰符

.stop 阻止事件冒泡
.prevent 阻止事件触发
.capture 优先触发
.self 只有自己能触发，子元素无法触发
.once 只能触发一次 

<style type="text/css">
   * {
       margin: 0 auto;
       text-align:center;
       line-height: 40px;
   }
   div {
       width: 100px;
        cursor:pointer;
   }
   #grandFather {
       background: deeppink;
   }
   #father {
       background: pink;
   }
   #me {
       background: hotpink;
   }#son {
        background: #ff4225;
    }
</style>

<div id="content">
    <div id="grandFather"  v-on:click="doc">
        grandFather
        <div id="father" v-on:click="doc">
            father
            <div id="me" v-on:click.stop="doc">
                me
                <div id="son" v-on:click="doc">
                son
            </div>
            </div>
        </div>
    </div>
<!--
1.在me上的click后面加一个 .stop， 那么冒泡到了这里就结束了，就不会冒到father上面去了
2.在father 上增加一个.capture<div id="father" v-on:click.capture="doc">那么当冒泡发生的时候，就会优先让father捕捉事件。点击son或者me的时候，都会优先触发它
3.修改father,增加 .self<div id="father" v-on:click.self="doc">这样点击son 和 me都不会导致其触发click事件，只有点击father自己，才会导致事件发生。
4.修改father,增加 .once<div id="father" v-on:click.once="doc">这样father点击一次之后，就不会再监听到click了
5.通过在 click 后面添加 .prevent 可以阻止页面刷新。注： 只有超链和form这种会导致页面刷新的操作，.prevent 才有用。 普通的不导致页面刷新的按钮，加上这个没有任何变化。
-->
</div>

</div>

<script>

      el: '#div1',

        name:"teemo"
      },

              alert(this.name);
          }
      }
    })
</script>
对应的修饰符：
.lazy 对于输入元素，默认的行为方式是一旦有数据变化，马上进行绑定。但是加上.lazy之后，相当于监听change操作，只有在失去焦点的时候，才会进行数据绑定了
.number 有时候，拿到了数据需要进行数学运算， 为了保证运算结果，必须先把类型转换为number类型，而v-model默认是string类型，所以就可以通过.number方式确保获取到的是数字类型了。
.trim 去掉前后的空白

<script>

new Vue({
      el: '#div1',
      data: {
          exchange:6.4,
          rmb:0,
          dollar:0
      },
      watch:{
          rmb(val) {
              this.rmb = val;
              this.dollar = this.rmb / this.exchange;
          },
          dollar(val) {
              this.dollar = val;
              this.rmb = this.dollar * this.exchange;
          },
      }

    })

</script>

区别：
computed是计算属性，也就是计算值，它更多用于计算值的场景；
computed具有缓存性，computed的值在getter执行后是会缓存的，只有在它依赖的属性值改变之后，下一次获取computed的值时重新调用对应的getter来计算
当一个属性受多个属性影响的时候就需要用到computed,最典型的例子： 购物车商品结算的时候

watch更多的是[观察]的作用，类似于某些数据的监听回调，用于观察props $emit或者本组件的值，当数据变化时来执行回调进行后续操作；
watch无缓存性，页面重新渲染时值不变化也会执行
当一条数据影响多条数据的时候就需要用watch,经典例子：搜索数据

十、filter过滤器

定义两个过滤器，分别是首字母大写和尾字母大写
然后通过如下方式同时首字母大小写

<style>
table tr td{
    border:1px solid gray;
    padding:10px;

}
table{
    border-collapse:collapse;
    width:800px;
    table-layout:fixed;
}
tr.firstLine{
    background-color: lightGray;
}
</style>

<div id="div1">

    <table align="center" >
        <tr class="firstLine">
            <td>输入数据</td>
            <td>过滤后的结果</td>
        </tr>      
        <tr>
            <td align="center">
                <input v-model= "data"  />
            </td>
            <td align="center">
                {{ data|capitalize|capitalizeLastLetter }}
            </td>
        </tr>
    </table>

</div>

<script>

new Vue({
      el: '#div1',
      data: {
          data:''
      },
      filters:{
          capitalize(value) {
                if (!value) return '' //如果为空，则返回空字符串
                value = value.toString()
                return value.charAt(0).toUpperCase() + value.substring(1)
          },
          capitalizeLastLetter(value) {
                if (!value) return '' //如果为空，则返回空字符串
                value = value.toString()
                return value.substring(0,value.length-1)+ value.charAt(value.length-1).toUpperCase()
          }
      }
    })

</script>

子组件给父组件传值，通过$emit();

<style>
div.product{
  float:left;
  border:1px solid lightGray;
  width:200px;
  margin:10px;
  cursor: pointer;
}
</style>

<div id="div1">
    <product @increment="changeTotal" @descrement="changeTotal">{{total}}</product>
</div>

<script>
components:{
      'product':{
          template:'<div class="product" >MAXFEEL休闲男士手包真皮手拿包大容量信封包手抓包夹包软韩版潮</div>',
          data(){
            return {
                count:0;
            }
          },
          methods:{
            increment(){
                this.$emit('increment',this.count++)
            },
            descrement(){
                this.$emit('increment',this.count--)
            }
          }
      }
    }

new Vue({
  el: '#div1',
  data:{
      total:0
  },
  methods:{
    changeTotal(counter){
        this.total=counter;//接受子组件传值
    }
  }
})
</script>
